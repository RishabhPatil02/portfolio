Missing alert notification system

Project information

  • Technologies: React, Spring Boot, MySQL, Multer, Axios
  • Languages: JS, Java
  • Source code:  
  • • Programmed a web application with React frontend and Spring Boot server, to connect users of common vicinity and notify them about any missing reports found in their area, through mail using the Gmail SMTP module.
  • • Constructed a database with SQL which enables upload and retrieval of missing victims’ images and last seen location.
