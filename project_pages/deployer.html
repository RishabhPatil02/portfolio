Scalable CI/CD Pipeline

Project information

  • Technologies: NextJS, Redis, ECS, ECR, S3, Docker, GitHub, Node, Express
  • Languages: JS
  • Source code:  
  • • Developed a scalable deployment architecture utilizing AWS ECS/ECR with a capability of running up to 5,000 Docker containers concurrently to clone GitHub repos and build executables, stored in AWS S3, hosted via a reverse proxy server.
  • • Built a Redis based publish-subscribe model to fetch real-time build logs via a socket server, displayed on NextJS client.
