COVID precaution system

Project information

  • Technologies: OpenCV, YOLO, SVM.
  • Languages: Python
  • Source code:  
  • Research paper:  IRJET
  • • Orchestrated a COVID precaution system using YOLOv4 model to detect human bodies, SVM classification and OpenCV library, to avoid crowd gatherings, maintaining social distance among people and mask detection on human faces
  • • Assembled a novel proportionality system by aggregating human body width, which could determine a threshold distance that should be maintained between two human bodies without using focal length of the camera
